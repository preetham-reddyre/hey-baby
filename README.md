# Hey Baby Project

## Project Documentation
This project is designed to help users manage their tasks efficiently with a user-friendly interface.

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/preetham-reddyre/hey-baby.git
   ```
2. Navigate into the project directory:
   ```bash
   cd hey-baby
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
